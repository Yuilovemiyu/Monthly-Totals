# 📊 Monthly Dashboard (Firebase + PWA)

This is a simple **Monthly Reporting Dashboard** connected to Firebase Firestore.

## 🚀 Features
- Live monthly totals (real-time Firebase sync)
- Grouped by month and store
- Auto-calculated totals (Total, Signed, Rejected, Cancelled)
- Bonus Calculator modal integration
- PWA support (installable + offline caching)
